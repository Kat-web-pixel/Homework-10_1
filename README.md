Проект "Обработка банковских транзакций"
Описание:
Проект "Обработка банковских транзакций" - это набор функций на Python для фильтрации и сортировки данных по транзакциям клиента.
Он предназначен для работы со списками транзакций, где каждая транзакция представлена в виде словаря с различными ключами,
например, 'date' и 'state'.
Основные функции позволяют фильтровать транзакции по их статусу (например, EXECUTED) и сортировать их по дате, что удобно
для аналитики и формирования отчетов.

pytest
Основные функции:
filter_by_state — фильтрует список операций по статусу (например, показывать только операции в статусе "Успешно").
sort_by_date — сортирует операции по дате (например, в начале списка будут отображаться самые последние операции).
Подготовьте список транзакций - список словарей с ключами 'date' и 'state': transactions = [
{'id': 1, 'amount': 500, 'state': 'EXECUTED', 'date': '2023-09-20'},
{'id': 2, 'amount': 1000, 'state': 'PENDING', 'date': '2023-09-19'},
{'id': 3, 'amount': 750, 'state': 'EXECUTED', 'date': '2023-09-21'},
{'id': 4, 'amount': 200, 'state': 'EXECUTED', 'date': '2023-09-18'},
]
Фильтруйте транзакции по статусу — например, оставить только выполненные:
executed_transactions = filter_by_state(transactions, 'EXECUTED')
Отсортируйте по дате — например, от самых свежих к более старым: 
sorted_transactions = sort_by_date(executed_transactions, reverse=True)
Выведите результат или используйте дальше по необходимости.

{'id': 1, 'amount': 500, 'state': 'EXECUTED', 'date': '2023-09-20'}
