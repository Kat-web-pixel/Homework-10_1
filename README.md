Проект создан для сортировки банковских операций клиента.
Установка:
Данный код не требует установки сторонних библиотек, поскольку использует только стандартные средства Python. Для начала, просто скопируйте функции в ваш проект.
Основные функции:
filter_by_state — фильтрует список операций по статусу (например, показывать только операции в статусе "Успешно").
sort_by_date — сортирует операции по дате (например, в начале списка будут отображаться самые последние операции).
Использование:
1. Подготовьте список транзакций - список словарей с ключами 'date' и 'state':
transactions = [
{'id': 1, 'amount': 500, 'state': 'EXECUTED', 'date': '2023-09-20'},
{'id': 2, 'amount': 1000, 'state': 'PENDING', 'date': '2023-09-19'},
{'id': 3, 'amount': 750, 'state': 'EXECUTED', 'date': '2023-09-21'},
{'id': 4, 'amount': 200, 'state': 'EXECUTED', 'date': '2023-09-18'},
]
2. Фильтруйте транзакции по статусу — например, оставить только выполненные:
executed_transactions = filter_by_state(transactions, 'EXECUTED')
3. Отсортируйте по дате — например, от самых свежих к более старым:
sorted_transactions = sort_by_date(executed_transactions, reverse=True)
4. Выведите результат или используйте дальше по необходимости.
Пример вывода:
Отфильтрованные и отсортированные транзакции:
{'id': 3, 'amount': 750, 'state': 'EXECUTED', 'date': '2023-09-21'}
{'id': 1, 'amount': 500, 'state': 'EXECUTED', 'date': '2023-09-20'}
{'id': 4, 'amount': 200, 'state': 'EXECUTED', 'date': '2023-09-18'}
